# Local Image Comparator Using AI 🤖

Offline Python desktop app for comparing two selected images and highlighting any detected changes. 🖼️

## Features 💫

* Fast local image alignment using ORB feature matching 🚀
* Local change detection using intensity and edge differences 🌟
* Highlighted overlay plus heatmap preview 🔥
* Exportable comparison report image 📥

## Run 🏃

Just open the exe file! ✨
